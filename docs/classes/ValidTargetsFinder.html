  • [settings] Object optional
  • [settings.limitToFov Bool

    = true] - If true only targets within this.entity's fov will be valid.

  • [settings.range Number

    = 1] - Max distance in tiles target can be from entity.

  • [settings.validTypes Array

    = []] - Array of valid target object types. Checked using target instanceof type.

  • [settings.includeTiles Bool

    = false] - If true tile objects are can be valid targets.

  • [settings.includeSelf Bool

    = false] - If true entity may target themself.

  • [settings.filter Bool

    = false] - * Function to filter objects when checking if they are valid. function(obj){ return true } . Targets must still be a valid type.
